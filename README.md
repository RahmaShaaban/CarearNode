# CareerNode 
**A Technical Career Guidance Platform for Computer Science Students**

##  Project Overview
CareerNode is a graduation project designed to help students navigate their academic paths (CS, IS, IT, etc.) through AI-powered CV analysis and interview simulations.

##  Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** Node.js, Express.js, Sequelize ORM
- **Database:** PostgreSQL (Supabase)
- **AI Module:** Python (NLP & Machine Learning)

##  Project Structure
- `backend/`: Express server, database models, and API routes.
- `frontend/`: User interface and client-side logic.
- `ai_module/`: Python scripts for CV parsing and recommendations.
